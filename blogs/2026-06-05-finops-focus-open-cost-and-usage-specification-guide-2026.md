FOCUS is an open standard that creates a unified schema for cloud cost data across AWS, Azure, Google Cloud, and other providers, allowing organizations to write a cost query once and run it against every provider. The specification, maintained by the FinOps Foundation, has reached version 1.3 as of December 2025 and is increasingly supported by major cloud vendors and SaaS platforms. Adopting FOCUS eliminates custom ETL pipelines and enables streamlined multi-cloud cost analysis and comparison.
